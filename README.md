# Rule-Based Chatbot (Python)

A simple rule-based chatbot developed in Python using dictionaries, functions, loops, and conditional statements. The chatbot responds to predefined user inputs such as greetings, jokes, help requests, and basic conversation prompts. It uses text preprocessing and keyword matching to provide appropriate responses and demonstrates the fundamentals of chatbot development without using AI or machine learning.

## Features

* User-friendly conversational interface
* Predefined responses using a knowledge base
* Randomized replies to avoid repetition
* Input cleaning and normalization
* Keyword and exact-match response handling
* Exit commands to end the conversation gracefully

## Technologies Used

* Python
* Dictionaries
* Functions
* Loops
* Conditional Statements
* Random Module

## Learning Objectives

This project demonstrates:

* Rule-based chatbot design
* String manipulation
* Dictionary usage
* Function implementation
* Basic Natural Language Processing concepts through keyword matching

## How to Run

1. Install Python.
2. Download the project files.
3. Run the Python file:
   python chatbot.py
4. Start chatting with the bot.
